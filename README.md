## Installation

### Prerequisites

-   [Node.js](https://nodejs.org/) and npm installed
-   [Expo CLI](https://docs.expo.dev/get-started/installation/) installed

### Clone Repository

```bash
git clone https://github.com/mirsahib/tuliptech-grid.git
cd tuliptech-grid
npm run install
npm run start
```
